Protocolo generador de interfaz:
Todas las peticiones son por post.
localhost:8080/nombre_modulo/action/

Acciones:

FillTable: Accion que carga los elementos del lista. 
PasteWindow: Accion que carga los elementos de las relaciones
Bring: Accion que obtiene los datos del elemento para editarlo.
Insert: Envio de los datos modelos para agregarlo
Update: Envio de los datos modelos para actualizar
Delete: Elimina un registro.
DeleteAll: Elimina un grupo de registros mandando sus IDs

Accion: FillTable y PasteWindow
Url: 
	localhost:8080/usuarios/fillTable/
	localhost:8080/usuarios/pasteWindow/	

Envia:
	model: Nombre del modelo
	per_page: Limite
	page: Offset
	order: order
	sort: sort
	filter: Elementos SQL agregados ejemplo:

Ejemplo: 
	 {model: “Permisos”, limit: 1, offset: 0, sort: “asc”, order: “name”, 
	 filter: {"where":[{"categories.is_offer IS NULL":null}]}

Devuelve: 
	{complete: bool, data: Array de Objetos, message: string, total: int }


Accion: Bring
Url: 
	localhost:8080/usuarios/bring/

Envia:
	filter: Elementos SQL agregados ejemplo:
	
Ejemplo:
	{filter: {"where":[{"categories.is_offer IS NULL":null}]}

Devuelve: 
	{complete: bool, data: Array de Objetos, message: string }







Action: Insert

Url: 
	localhost:8080/usuarios/insert/

Envia:
	Nombre del campo del modelo: Valor del campo

Ejemplo: 
	{name: “Omar”, first_name: “Sarmiento”: age: 27}

Devuelve: 
	{complete: bool, data: Array de Objetos, message: string }

Action: Update

Url: 
	localhost:8080/usuarios/update/

Envia:
	Nombre del campo del modelo: Valor del campo
	Id: Campo llave primario

Ejemplo:
	 {id: 12, name: “Omar”, first_name: “Sarmiento”: age: 27}

Devuelve: 
	 {complete: bool, data: Array de Objetos, message: string }



Action: Delete

Url: 
	localhost:8080/usuarios/delete/

Envia:
	Nombre del campo del modelo: Valor del campo
	Id: Campo llave primario

Ejemplo:
	 {id: 12, name: “Omar”, first_name: “Sarmiento”: age: 27}

Devuelve: 
	 {complete: bool, data: Array de Objetos, message: string }





Action: DeleteAll

Url: 
	localhost:8080/usuarios/deleteAll/

Envia:
	Id: Campo llave primario separados por comas

Ejemplo:
	 {id: 12,1,34}

Devuelve: 
	 {complete: bool, data: Array de Objetos, message: string }
